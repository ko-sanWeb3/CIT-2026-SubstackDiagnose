# 自分史は
# 公開しなくて大丈夫

誰かに見せるためではありません。

まずは

- 自分の記録
- 自分の思い出
- 自分の人生

を書き残しましょう。

---

# テーマは
# ひとつだけ選ぶ

たとえば

- 昔の仕事
- 旅の思い出
- 家族に伝えたい話

どれかひとつで十分です。

---

# まず3行だけ書く

Substackで「新規投稿」を開き、

1. タイトルを書く
2. 3行だけ書く
3. 下書き保存する

上手に書く必要はありません。
